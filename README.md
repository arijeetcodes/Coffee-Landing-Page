## 📌 Features

* 🎨 Clean and modern UI design
* 📱 Fully responsive layout
* ⚡ Smooth scroll animations using AOS
* ⌨️ Custom **typewriter effect** in hero section
* 🎯 Interactive hover effects & transitions
* 🧩 Reusable components (cards, buttons, sections)
* 🎛️ CSS variables for consistent theming

---

## 🛠️ Tech Stack

* **HTML5** – Structure & layout
* **CSS3** – Styling, Flexbox, animations, responsiveness
* **JavaScript** – Typewriter effect & interactivity
* **AOS Library** – Scroll animations
* **Font Awesome** – Icons

---

## 📚 What I Learned

* Structuring a complete landing page using semantic HTML
* Mastering **Flexbox** for layout design
* Using **CSS variables (:root)** for scalable styling
* Creating smooth **hover effects, transitions, and transforms**
* Implementing **scroll-based animations (AOS)**
* Building a **typewriter animation from scratch using JavaScript**
* Improving UI/UX with micro-interactions and visual hierarchy
* Making layouts responsive using **media queries**
